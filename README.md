# KINETIC — Prototipo web (HTML, CSS, JavaScript)

Prototipo funcional con renderizado dinámico desde JSON

## Ver el proyecto en el navegador

Abre esta URL (el sitio debe servirse por **HTTP**, no abriendo el `.html` con doble clic):

**[LINK APP](https://damarislh10.github.io/KINETIC-8/)**


## Páginas principales

| Ruta | Descripción |
|------|-------------|
| `index.html` | Inicio, servicios desde JSON, testimonios, contacto |
| `catalogo.html` | Catálogo con filtros y favoritos |
| `servicio.html?id=<id>` | Detalle de un servicio |
| `admin.html` | Acceso admin (formulario validado, demo) |
| `nosotros.html` | Misión, principios y equipo (JSON) |

## Requisitos

- Navegador actual (Chrome, Edge, Firefox).
- **Servidor HTTP local** (obligatorio para `fetch` del JSON y módulos ES6).

## Datos

- `data/services.json` — servicios, testimonios, principios y equipo.
